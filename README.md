# 🕊️ Gerador de Versículos Bíblicos para Holyrics

Uma aplicação web simples, elegante e responsiva para gerar versículos bíblicos formatados perfeitamente para uso no [Holyrics](https://holyrics.com.br/).

## ✨ Recursos

- **Múltiplas Versões da Bíblia**: Suporte para ARA, ACF, NVI e NVT.
- **Interface Dark Moderna**: Design focado em usabilidade com efeitos *glassmorphism*, texturas e animações.
- **Pesquisa Inteligente**: Caixas de seleção customizadas com busca rápida pelo nome do livro.
- **Múltiplos Blocos**: Adicione quantos blocos de versículos quiser de diferentes livros e capítulos de uma só vez.
- **Exportação Rápida**: Copie tudo para a área de transferência com um clique ou baixe um arquivo `.txt` pronto para colar.
- **Tutorial Integrado**: Passo a passo acessível de como usar o resultado gerado diretamente na letra da música no Holyrics.

## 🚀 Como usar

1. Abra o `index.html` no seu navegador.
2. Selecione a versão da Bíblia desejada no topo (ARA, ACF, NVI ou NVT).
3. Clique em **Adicionar Versículo** para criar um novo bloco.
4. Escolha o Livro, Capítulo e os Versículos (Inicial e Final).
5. Clique em **Gerar Resultado**.
6. Clique em **Copiar Tudo** ou **Baixar .txt**.
7. No Holyrics, crie uma nova música, defina o fundo como "Bíblia" e cole o resultado no campo de letra.

## 🛠️ Construído com

- **HTML5** & **CSS3** (Variáveis CSS, Flexbox, Grid, Animações)
- **JavaScript** puro (Vanilla, manipulação de DOM, fetch API)
- Dados das Bíblias em formato JSON extraídos e consumidos dinamicamente de repositórios do GitHub.
